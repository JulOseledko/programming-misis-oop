# Programming MISIS OOP

Курс: ООП
НИТУ МИСИС, 4 семестр.

Проект представляет собой небольшую RPG-игру на Java с использованием LibGDX. В ходе работы были реализованы базовые игровые механики: управление героем, монстры, атаки, здоровье, карта с препятствиями, предметы, опыт, HUD, анимация, камера и сборка запускаемого `.jar`-файла.

## Стек

- Java
- LibGDX
- Gradle
- ООП: классы, наследование, инкапсуляция, абстракция, композиция

## Структура

```text
core/src/       основная игровая логика
core/assets/    графика, шрифты и игровые ресурсы
desktop/src/    desktop launcher
release/        собранный jar-файл
```

## Запуск

```bash
java -jar release/game.jar
```
